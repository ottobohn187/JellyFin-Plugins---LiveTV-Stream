# JellyFin Plugins - LiveTV Stream

Workspace for the Jellyfin LiveTV Stream plugin and related development notes.

## Jellyfin Repository URL

Add this URL in Jellyfin under `Dashboard -> Plugins -> Repositories`:

`https://raw.githubusercontent.com/ottobohn187/JellyFin-Plugins---LiveTV-Stream/main/manifest.json`

After the GitHub Action runs, Jellyfin can read that manifest and install the packaged plugin directly from GitHub.
